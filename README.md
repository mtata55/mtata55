**Writing:**
- 📚 [LLMs and the Future of Cognition](https://meheryar.substack.com/s/llms-and-the-future-of-cognition) — series on the *valido problem*: the 2,400-year-old pattern of a single trusted advisor capturing the decision-maker, now reborn as the LLM as perfect valido — and what adversarial cooperation looks like as the operator's response
- 🔁 [The Founder Leverage Loop](https://meheryar.substack.com/s/the-founder-leverage-loop) — series on phase-awareness in founder behavior. The same hands-on work that generates learning early stops doing so at predictable transition points, and continuing past those points quietly destroys leverage. The loop: hands-on effort → encoded into systems → systems generate signal → signal triggers targeted hands-on → repeat

**Currently building:**
- **[Riff](https://getriff.org)** — a collaborative thinking tool for human+AI pairs. The working implementation of the adversarial-cooperation thesis: human-to-human collaboration is slow, but human+AI to human+AI is exponentially faster *if* the system enforces friction against premature coherence. The decision trace it produces as a byproduct (Talmud pattern: the spec is the Torah, the discussion log is the centuries of argument) is more valuable than the spec itself. Source: [`mtata55/spec-collab`](https://github.com/mtata55/spec-collab) (currently private).
